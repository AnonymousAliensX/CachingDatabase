# CachingDatabase

Usage : 

  Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
  Add the dependency in app gradle :
  
	dependencies {
	        implementation 'com.github.AnonymousAliensX:CachingDatabase:1.01'
	}
  
  
  Enjoy :)
